# Vagrant

### install Ubuntu

precise32: an image for a VPS with Ubuntu 12.04 operating system

    vagrant box add precise32 http://files/vagrantup.com/precise32.box
    
    # create a project
    mkdir test_project
    cd test_project
    
    vagrant init
    
modify Vagrantfile

    nano Vagrantfile
    
    config.vm.box = "base"  --> config.vm.box = "precise32"
    
    ctrl+O -> save
    ctrl+X -> exit

under the `test_proejct`

    vagrant up
    vagrant ssh
