# Git CodeSchool

### Level 1
to get help
```
git help
git help config
```
to create a user for all repos
```
git config --global user.name "XX"
git config --global user.email xxx@xx.com
git config --global color.ui true
```
to creat a new repo
```
mkdir store
cd store
git init
```
to check the status
```
git status
```
to add a file
```
git add index.html
git commit -m "new file index.html created"
```
