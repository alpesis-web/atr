# Vim

## 1. Basic

<b>Normal Mode</b>  
edit  
- to insert: `i` 
- to delete: `x`
- to exit: `esc`

move  
- to move: `hjkl`, h - left, l - right, j - down, k - up
- (move) back word: `b`
- (move) next word: `w`
- (move) end of line: `$`
- (move) page up/down: `ctrl + u/d`
- (move) top of the file: `gg`

copy, paste, change   
- to yank/copy: `y(xxx)`
- (yank) current line: `yy`
- to paste: `p`
- to change: `c`
- (change) word: `cw`

<b>Command Mode</b>  
- to write/save: `:w file_path`  
- to open file: `:e file_path`
