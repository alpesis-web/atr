# Emacs Commands

## 1. Basic Commands

C-x: `ctrl-x`
M-x: `Alt-x`

close emacs: `ctrl-x, ctrl-c`  
help: `ctrl-h`  
- the info directory: `ctrl-h i`
- menu: `m`  
- quit: `ctrl-g`
- up: `u`
- back to the last page: `l`
- search: `ctrl-s`

## 2. Customization and Themes

- find documents: `C-x, C-f`
