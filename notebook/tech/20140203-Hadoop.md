# Hadoop

### 1. Hadoop and SQL Database

Comparison of Hadoop and SQL Database  

| Hadoop                             | SQL Database              |
|:-----------------------------------|:--------------------------|
| scale-out                          | scale-up                  |
| key/value                          | relational tables         |
| functional programming (MapReduce) | declarative queries (SQL) |
| offline batch processing           | online transactions       |
