## 1. Flatlander's Gem Store

server (data store <-> api) - developers / mobile app / browser app (angular.js)

library, app name, dependencies
```js
var app = angular.module('store', [])
```

example
```js
var app = angular.module('gemStore', []);
```

```html
<!DOCTYPE html>
<html ng-app='gemStore'>
  <head>
```
