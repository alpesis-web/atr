# Predictive Modeling for Sports Games

### 1. Football

prediction = historicalKicks + recentStatus

kickRate = ( total # of kicks + # of kicks of recent fives) / # of games

### 2. Basketball

winProb = 0.5 + 0.03 * diffRanks
